# GRC Projects
A collection of GRC-related projects used to strengthen the skills necessary for a GRC professional
## Risk Register

This project presents a structured cybersecurity risk register developed for a fictional healthcare organization, showcasing a practical, repeatable approach to risk management aligned with industry GRC practices. It includes a defined risk acceptance matrix and ten documented security and operational risks, each evaluated using inherent and residual impact and likelihood scoring. Every entry outlines a clear risk scenario, assigned ownership, treatment strategy, and business-focused justification, with controls mapped to frameworks such as NIST SP 800-53, ISO/IEC 27001, and the HIPAA Security Rule. The result is a concise, auditable artifact that demonstrates core competencies in qualitative risk analysis, control mapping, and translating technical threats into business-relevant risk insights for governance, compliance, and decision-making.
